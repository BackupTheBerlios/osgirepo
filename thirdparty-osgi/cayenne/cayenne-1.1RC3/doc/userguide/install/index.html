2. Installation


2.1 System Requirements

Cayenne, including GUI tools, is written in 100% Java, and therefore should run out of the box on any Java-compatible platform. More detailed requirements are listed below:

- JDK. Cayenne applications can be deployed under JDK version 1.3 or higher. JSDK 1.4 or higher is required for development GUI tools. Download JDK from Sun at http://java.sun.com/products/

- Java Compatible Operating System.

 Mac OS X Note:
Make sure that Java is installed on HFS+ partition. Swing and AWT do not work from UFS!

- Access to a relational database and JDBC driver for this database. Out of the box Cayenne supports a number of databases. If your database is not on the list, there is a very good chance you can still use Cayenne with it, as long as there is a JDBC driver available. List of available JDBC drivers can be found at http://industry.java.sun.com/products/jdbc/drivers.

- XML SAX Parser (JDK1.3 only). If the system runs JDK1.3, a third party SAX parser is required (JDK1.4 has a parser already bundled with it). Cayenne was tested with Apache Xerces parser version 2.02. To install Xerces download it from http://xml.apache.org/, unpack the distribution, and then copy xercesImpl.jar and xmlParserAPIs.jar into the JRE extensions directory ($JRE_HOME/lib/ext).

- JDBC2.0 Optional Package (JDK1.3 only). If the system runs JDK1.3, JDBC extensions package is required (javax.sql). It can be downloaded from Sun at http://java.sun.com/products/jdbc/download.html#spec and copied to the JRE extensions directory ($JRE_HOME/lib/ext).

- (Optional) Ant 1.4.1 or newer. Ant can be downloaded from http://jakarta.apache.org/ant/index.html. Ant is needed for:

  • cgen Ant task for class generation. An alternative to cgen is to use CayenneModeler GUI tool.
  • Building Cayenne from source, for instance when changing Cayenne source code to customize it for your project.

- (Optional) JavaGroups. If you plan to use remote notifications via JavaGroups transport, javagroups-*.jar must be available at runtime. It can be downloaded from here. Cayenne has been tested with JavaGroups 2.1.1.

- (Optional) JMS. If you plan to use remote notifications via JMS transport, J2EE libraries must be available at runtime, and JMS services must be running.

- (Optional) JDOM. If you develop a Swing application that uses DataViews, you need to have JDOM jars installed. JDOM can be downloaded from http://www.jdom.org/.
